# 《迟到焰火》故事总控页

## 项目基础

| 项目 | 内容 |
| --- | --- |
| 暂定书名 | 《迟到焰火》 |
| 一句话卖点 | 五年后，危机公关林晚意被派回海边旧酒店做九十天翻盘项目，必须和当年亲手放弃她的前任周砚城一起，在焰火季前救下一家快被资本吞掉的老酒店。 |
| 题材 | 现代都市言情 |
| 子类型 | 破镜重圆 / 双强拉扯 / 成年人恋爱 / 职场合作对抗 |
| 预计篇幅 | 中长篇 |
| 叙事视角 | 双视角为主，以女主视角开篇 |
| 时态与语言风格 | 第三人称过去时感受式叙述，语言克制、锋利、带海风质感 |
| 目标读者 | 喜欢现代都市拉扯、破镜重圆、双强合作、情绪兑现型言情的读者 |
| 核心情绪体验 | 克制、拉扯、旧情复燃、互相试探、并肩作战、代价式圆满 |
| 结局承诺 | HE，但需要明确付出代价后才走到一起 |

## 故事承诺

### 1. 读者为什么会点开

- 海边旧酒店翻盘与焰火季倒计时，天然自带场景感和紧迫感。
- 前任重逢不是偶遇，而是被迫一起救同一个项目，无法回避。
- 两个人都很能做事，也都很会把最想说的话咽回去，情绪张力强。

### 2. 读者为什么会继续追

- 九十天倒计时会持续制造现实压力。
- 每一阶段都有新危机：翻盘、舆论、内部背刺、旧债、选择站队。
- 关系推进不是凭空发糖，而是在一次次共同解决问题中被逼出来。

### 3. 这本书绝对不能写偏成什么

- 不能写成只有装修和项目会的职场流水账。
- 不能靠偷听、失联、装瞎式误会硬拖剧情。
- 不能把主角写成只会嘴硬、不承担后果的人。

## 核心冲突

### 外部冲突

- 主线问题：栖岸酒店能不能在九十天内拿下临屿焰火季主会场资格，避免被打包出售。
- 客观障碍：酒店老旧、舆论危机、董事会短视、竞争对手压价、内部资金链紧张。
- 逼迫两人持续接触或对立的机制：林晚意代表曜石文旅执行翻盘项目，周砚城是酒店实际运营负责人，任何决定都绕不过对方。

### 内部冲突

- 女主内在缺口：她相信离开和自保，胜过相信承诺和留下。
- 男主内在缺口：他总把责任放在第一位，默认自己配不上“先选自己”。
- 关系的核心障碍：他们都曾在对方最需要的时候没有真正站到对方身边，因此谁都不肯先承认还想要。

### 关系不能立刻成立的原因

- 硬障碍：立场相对、旧创未愈、酒店未来不确定、女主升职机会和项目结果直接绑定。
- 软障碍：自尊、防御机制、旧账未清、两个人都习惯把真实需求包进工作里。

## 必留设定

| 设定项 | 内容 | 不写会损失什么 |
| --- | --- | --- |
| 时代/背景 | 当代沿海虚构城市“临屿” | 缺少真实的都市压力和节奏感 |
| 核心场景 | 栖岸酒店，一家建在海崖边的旧酒店 | 会失去故事的独特氛围和记忆锚点 |
| 社会规则/身份差 | 文旅资本与地方旧业态的碰撞 | 关系冲突会失去现实抓手 |
| 利益结构 | 曜石文旅、酒店管理层、竞争收购方三方博弈 | 项目线无法成立 |
| 禁忌或边界 | 不能随意裁掉老员工，不能轻易卖掉屋顶观景层 | 失去男主坚持和价值观冲突来源 |

## 主线引擎

### 这本书的推进方式

- 关系推进主引擎：前任重逢后的合作对抗、共同救场、代价性交心。
- 剧情推进主引擎：九十天翻盘项目、舆论危机、内部背刺、焰火季竞标。
- 节奏主引擎：每章有变化、每段有兑现、章末留钩子。

### 三个必须兑现的大场面

1. 暴雨断电夜，林晚意和周砚城第一次并肩救场，旧默契强行回归。
2. 屋顶试放冷焰火时，两人终于承认旧账没过去，却又舍不得彻底翻脸。
3. 焰火季终场，周砚城公开做出割舍，林晚意也第一次不是为了体面而离开。

## 主要人物

### 林晚意

- 年龄：29
- 身份：曜石文旅危机项目负责人
- 人设关键词：冷静、利落、清醒、嘴硬、执行力极强
- 最大优势：判断快，控场强，能在混乱里立刻找到切口
- 最大弱点：太会体面地退出，不肯在感情里示弱

### 周砚城

- 年龄：31
- 身份：栖岸酒店实际运营负责人
- 人设关键词：克制、稳、能扛、寡言、手上有活
- 最大优势：能把人和事都稳住，关键时候会顶上去
- 最大弱点：习惯替所有人做决定，把自己想要的放最后

### 许茵

- 身份：栖岸酒店甜品主厨，林晚意旧友
- 功能：给故事温度和生活质感，也负责看穿两位主角的自我欺骗

### 顾逢舟

- 身份：曜石文旅 COO
- 功能：现实压力化身，代表“结果优先”的资本逻辑

### 周聿宁

- 身份：栖岸酒店新媒体负责人，周砚城表妹
- 功能：制造信息流动、网络视角和年轻表达

### 孟其修

- 身份：竞争收购方潮湾文旅负责人
- 功能：外部威胁，推动男主和女主尽快做选择

## 文风与边界

### 文风关键词

- 语言气质：冷感、锋利、留白多
- 叙述密度：中速偏快
- 对话风格：不废话，带刺，成年人式试探
- 情绪表达：克制，不直接喊痛，但读者能看见疼

### 禁写项

- 不写悬浮台词：禁止“你是我的全世界”这类空泛表白。
- 不写硬拗误会：禁止靠偷听半句、关机失联、故意不解释制造长篇误会。
- 不写角色降智：主角不为剧情需要故意做蠢决定。
- 不写灌水信息：酒店运营和项目信息只写到足够支撑冲突为止。

### 亲密戏与尺度边界

- 可写范围：拥抱、亲吻、情绪推进型亲密场面。
- 不可写范围：脱离情绪和关系推进的纯动作描写。
- 情绪目标：亲密戏必须推动关系真相，不是拿来冲字数。

## 开篇承诺

- 认识谁：林晚意、周砚城、许茵、顾逢舟。
- 看到什么冲突：翻盘项目、旧情重逢、立场对撞。
- 对哪段关系起兴趣：林晚意和周砚城为什么明明默契还要硬顶着对方。
- 想追问什么：五年前到底是谁先放手，这次他们还能不能不重蹈覆辙。

## 写作警报

- 如果章节只在说项目，没有让人物关系发生变化，就返工。
- 如果周砚城变成只会沉默的木头，返工。
- 如果林晚意变成只会嘴硬的模板女主，返工。
- 如果任何冲突离开“留下还是离开”这个核心问题也能成立，返工。
